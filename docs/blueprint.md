# **App Name**: VendorFlow

## Core Features:

- Personalized Onboarding: Guides new users through an initial setup, prompting for 'Business Name,' 'Owner Name,' 'Location,' and an optional 'Business Profile Picture' to customize their experience.
- Business Dashboard: A prominent home section displaying the user's business name, owner's name, location, and uploaded business profile image.
- Product Management: Allows vendors to add, edit, and categorize an unlimited number of products, each with multiple image uploads and detailed descriptions.
- AI Product Description Tool: Utilizes generative AI as a tool to help vendors create engaging and unique descriptions for their products based on input keywords and attributes.
- Customer Management: Track customers by name, phone number, and an optional profile photo for easy lookup and personalization.
- Transaction Tracking: Enables creation of new transactions, listing items and quantities, and meticulously tracking 'pending' vs. 'paid' amounts with support for partial payments.
- Local Data Persistence: Securely stores all application data (business, products, customers, transactions) locally within the user's browser, interpreting the request for 'text files' and 'in-app storage' via client-side persistent storage mechanisms.

## Style Guidelines:

- Primary color: Professional Blue (#1670CC). This color signifies trustworthiness and efficiency, setting a confident and clean tone for the business platform.
- Background color: Subtle Pale Grayish-Blue (#F3F6F8). This provides a serene and spacious backdrop that enhances readability and keeps the focus on content without being stark white.
- Accent color: Vibrant Cyan-Aqua (#66DBE4). Positioned analogous to the primary color, this accent adds a modern, energetic touch while maintaining a professional appearance, ideal for calls to action or highlights.
- Body and headline font: 'Inter' (sans-serif). Chosen for its contemporary, neutral, and highly readable qualities, it ensures clarity and professionalism across all data and textual content.
- Clean, modern line icons: Simple, vector-based line icons will provide a consistent and intuitive visual language that complements the professional aesthetic without adding unnecessary visual clutter.
- Grid-based dashboard layout: A structured, modular layout will ensure easy navigation and clear organization of business data, allowing vendors to quickly access and understand key information.
- Subtle, functional transitions: Discrete and purposeful animations will provide responsive feedback and guide user attention without being distracting, enhancing the seamless user experience.
